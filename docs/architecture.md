## Технологии
| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Бэкенд | Flask | Обработка запросов |
| Шаблоны | Jinja2 | Генерация HTML |
| Хранение | JSON | Сохранение данных |
| Стили | CSS | Внешний вид |

## Маршруты
| Маршрут | Метод | Описание |
|---------|-------|----------|
| `/` | GET | Главная страница |
| `/add` | GET/POST | Добавление желания |
| `/wish/<id>` | GET | Детальный просмотр |
| `/edit/<id>` | GET/POST | Редактирование |
| `/delete/<id>` | GET | Удаление |
| `/toggle/<id>` | GET | Переключение статуса |
| `/search` | GET | Поиск по названию |
| `/filter/<status>` | GET | Фильтр по статусу |
