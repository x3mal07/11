# Требования к проекту

## Проблема
Люди часто забывают, что хотели бы купить, или теряют ссылки на товары. Приложение помогает систематизировать желания.

## Целевая аудитория
Люди, которые хотят планировать свои покупки и ничего не забыть.

## Функциональные требования
1. Добавление желания (название, описание, цена, ссылка, статус)
2. Просмотр всех желаний
3. Детальный просмотр одного желания
4. Редактирование желания
5. Удаление желания
6. Поиск по названию
7. Фильтр по статусу (Хочу / Куплено)
8. Подсчёт общей стоимости
9. Переключение статуса (Хочу → Куплено)

## Архитектура
Браузер → Flask → JSON-файл
