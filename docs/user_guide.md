# Руководство пользователя

## Как пользоваться приложением

### Главная страница
- Показывает список всех желаний
- Видна общая стоимость всех желаний
- Кнопка "Добавить желание" → переход к форме

### Добавление желания
Заполните поля:
- **Название** — обязательно
- **Описание** — необязательно
- **Цена** — необязательно
- **Ссылка** — необязательно
- **Статус** — Хочу / Куплено

### Управление желаниями
- **✏️** — редактирование
- **🗑️** — удаление
- **✅ Купить / ↩️ Вернуть** — переключение статуса

### Поиск и фильтры
- Введите текст в поле поиска → поиск по названию
- Нажмите "Хочу" или "Куплено" → фильтрация
