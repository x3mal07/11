# 🎁 Список желаний

Веб-приложение для создания и управления списком желаний.

## Возможности

- ➕ Добавление желаний (название, описание, цена, ссылка)
- ✏️ Редактирование и удаление
- 🔍 Поиск по названию
- 📊 Фильтр по статусу (Хочу / Куплено)
- 💰 Подсчёт общей стоимости
- ✅ Отметка о покупке

## Запуск

```bash
pip install flask
python app.py
